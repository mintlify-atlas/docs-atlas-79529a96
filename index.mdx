<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">
    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get DevPush running on your server in minutes
  </p>

  <Steps>
    <Step title="Install on your server">
      Run the installation script on a fresh Ubuntu/Debian server with SSH access:

      ```bash
      curl -fsSL https://install.devpu.sh | sudo bash
      ```

      The installer sets up Docker, creates the `devpush` system user, clones the repository, and configures systemd.
    </Step>

    <Step title="Create a GitHub App">
      DevPush uses a GitHub App for authentication and repository access. Follow the [GitHub App setup guide](/guides/github-app) to create one.

      You'll need to configure these settings in `/var/lib/devpush/.env`:
      - `GITHUB_APP_ID`
      - `GITHUB_APP_NAME`
      - `GITHUB_APP_PRIVATE_KEY`
      - `GITHUB_APP_WEBHOOK_SECRET`
      - `GITHUB_APP_CLIENT_ID`
      - `GITHUB_APP_CLIENT_SECRET`
    </Step>

    <Step title="Configure and start">
      Edit `/var/lib/devpush/.env` with your domain settings and start the service:

      ```bash
      sudo systemctl start devpush.service
      ```

      Set DNS records for your app hostname and deployment domain (wildcard A record).
    </Step>

    <Step title="Deploy your first project">
      Log in to your DevPush instance, connect your GitHub account, and create your first project. Push to your repository and watch your deployment go live with zero downtime.
    </Step>
  </Steps>
</div>
